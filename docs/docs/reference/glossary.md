# Glossary

| Term | Meaning |
|------|---------|
| **Gate** | PR-oriented review mode with check run / comments |
| **Stewardship** | Branch/path audit mode with durable findings |
| **Unit** | Planned batch of files for specialists |
| **Specialist** | Role-focused agent (security, correctness, …) |
| **Discourse** | Thorough dual-pass dialogue among specialists |
| **Judge** | Final keep/drop/severity aggregation |
| **Finding** | Durable issue with fingerprint and lifecycle |
| **Learning** | Org memory from reactions / dismissals |
| **Graph** | Structural code index via Codesteward Graph MCP |
| **Platform operator** | Install-wide admin (not just org admin) |
| **SoT** | Source of truth (Postgres for jobs) |
